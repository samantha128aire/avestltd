# AGENTS.md - Your Workspace

## Session Start
1. Read `SOUL.md` — who you are
2. Read `USER.md` — who you're helping  
3. Read `IDENTITY.md` — your identity
4. Read `memory/YYYY-MM-DD.md` (today + yesterday)
5. **If MAIN SESSION** (direct chat): Also read `MEMORY.md`

## Memory
- **Daily logs:** `memory/YYYY-MM-DD.md` — raw notes of what happened
- **Long-term:** `MEMORY.md` — curated memories (MAIN SESSION ONLY, contains personal data)
- **Project memory:** `memory/projects/<project>/` — per-project goals, decisions, status
  - `avestai/` — goals.md, decisions.md, status.md
  - `propsuav/` — status.md
  - `skyranger/` — status.md
  - `mooney/` — status.md
- **CRM:** `~/.openclaw/workspace/data/avestai.db` — SQLite (leads, contacts, customers, revenue)
- **Write it down immediately** — if you want to remember something, write it to a file. "Mental notes" don't persist.

## Safety
- No data exfiltration, ever
- `trash` > `rm` (recoverable beats gone)
- Ask before external actions (emails, posts, tweets)

## Heartbeats
Follow `HEARTBEAT.md`. Reply `HEARTBEAT_OK` if nothing needs attention.

Use heartbeats for batch checks that can drift. Use cron for exact timing or isolated tasks.

## Proactive Work (During Heartbeats)
- Read and organize memory files
- Check project status
- Update documentation
- Review and update MEMORY.md periodically

## Skills
Check `SKILL.md` files when you need specific tools. Keep local notes in `TOOLS.md`.
