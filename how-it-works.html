From Order to AI Assistant in 7 Days

We've streamlined our process to deliver your fully-configured OpenClaw AI assistant as quickly as possible — without cutting corners.

0

Discovery

You fill out our detailed intake form (10-15 minutes)

We learn about:

  • Your daily workflows & pain points
  • Accounts you want integrated (email, calendar, etc.)
  • Industry-specific needs (aviation, business, agriculture)
  • Preferred AI personality (formal, casual, proactive, reactive)
  • Smart home devices & Apple ecosystem
  • Privacy preferences & security concerns

We review your form and reach out (within 24 hours) to:

  • Confirm details
  • Answer any questions
  • Provide payment link (Stripe secure checkout)
1

Hardware Preparation

Standard/Premium customers:

  • We order your Mac Mini (M2 or M2 Pro)
  • Unit arrives at our facility (typically next-day)
  • Inspect for any shipping damage
  • Document serial number

Basic customers:

  • You ship us your Mac Mini (we provide prepaid shipping label)
  • We inspect upon arrival, document condition
  • Notify you of receipt & begin setup
2-4

Configuration & Setup

System Preparation (4-6 hours):

  • Unbox & test Mac Mini
  • Install latest macOS updates
  • Configure system settings (energy, security, privacy)
  • Enable FileVault disk encryption
  • Configure firewall & network settings
  • Create user accounts (customer account + support account)

OpenClaw Installation (3-5 hours):

  • Install Homebrew, Node.js, OpenClaw
  • Configure AI provider (Anthropic/OpenAI/Google per your choice)
  • Set up API keys (we help you create accounts if needed)
  • Create workspace structure
  • Write persona files (USER.md, IDENTITY.md, SOUL.md, TOOLS.md)
  • Configure memory system

Integration Setup (3-5 hours):

  • Connect email accounts (Gmail, iCloud, Outlook)
  • Configure calendar sync
  • Set up messaging (iMessage, WhatsApp, etc.)
  • Connect cloud storage (Google Drive, Dropbox)
  • Configure task management tools
  • Install industry-specific skills (aviation, business, etc.)
  • Smart home integration (HomeKit, Sonos, etc.)

Security Configuration (1-2 hours):

  • Set up weekly security health reports (sent to our monitoring system)
  • Configure automatic backups (to your cloud storage)
  • Install remote access tools (Chrome Remote Desktop + TeamViewer backup)
  • Test security settings
5

Testing & QA

Comprehensive Testing (2-3 hours):

  • OpenClaw gateway running & responsive
  • Email reading, searching, drafting
  • Calendar event creation, queries, reminders
  • Messaging send/receive (iMessage, etc.)
  • Memory system (save/recall information)
  • All installed skills functional
  • Heartbeat tasks triggering correctly
  • Smart home commands (if applicable)

Security Audit:

  • Firewall enabled & configured correctly
  • FileVault encryption active
  • Remote access secure (PIN set, 2FA enabled)
  • No unnecessary ports open
  • System updates current

Performance Benchmarking:

  • CPU/RAM usage acceptable (<50% idle)
  • Response time <5 seconds
  • No error messages in logs

Documentation Preparation:

  • Write handoff document (personalized to your setup)
  • Create quick reference guide
  • Prepare FAQ based on your specific configuration
6

Pre-Shipment Remote Connection Test ⭐

This is our secret sauce!

Before we ship your Mac Mini, we schedule a 15-30 minute video call to:

1. Walk you through remote access:

  • Open Chrome on your device (laptop, iPad, phone)
  • Go to remotedesktop.google.com
  • Sign in with your Google account
  • Connect to your Mac Mini (we guide you step-by-step)

2. Verify everything works:

  • Test connection quality
  • Show you OpenClaw responding to commands
  • Demonstrate key integrations (email, calendar, etc.)

3. Address any concerns:

  • Answer questions
  • Make last-minute adjustments if needed
  • Confirm you're comfortable before shipping

Why we do this:

  • Catches any issues while we still have the Mac Mini
  • Builds your confidence ("I've already used it!")
  • Reduces post-delivery support needs
  • Ensures you're ready to use it the moment it arrives
7

Shipping & Handoff

Secure Packaging:

  • Mac Mini safely packaged in original Apple box + outer shipping box
  • Includes: Power cable, setup guide (printed), quick reference card, support contact info
  • Fully insured shipping (FedEx or UPS, signature required)

You receive:

  • Tracking number (emailed immediately)
  • Handoff documentation (PDF emailed)
  • Onboarding call scheduling link (Calendly)

Estimated delivery: 2-3 business days (US domestic)

📞

Post-Delivery: Onboarding Call

Scheduled within 24-48 hours of delivery

Duration: 1-2 hours (longer for Premium customers)

Agenda:

  1. System Overview (15 min):

    Power on procedure, where OpenClaw "lives" and how it works, network requirements, basic troubleshooting

  2. Remote Access Confirmation (10 min):

    You demonstrate connecting (since you already did this pre-shipping!), verify any issues resolved, show mobile access

  3. OpenClaw Walkthrough (30 min):

    How to interact, example queries & commands, memory system explanation, heartbeat features, voice/tone customization

  4. Integration Demos (20 min):

    Email management, calendar handling, reminders & tasks, smart home, industry-specific features

  5. Security & Privacy (10 min):

    What data OpenClaw has access to, how to adjust permissions, remote access explanation, how to disable/enable features

  6. Support Information (10 min):

    How to reach us, 30-60 day support scope, response time expectations, common issues & solutions, support incentive explanation

  7. Q&A (remaining time)
