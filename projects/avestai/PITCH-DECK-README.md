## Tips for Delivery
- **Keep it conversational** — This is networking, not a boardroom pitch
- **Focus on outcomes** — "Saves 15 hours/week" not "uses OpenClaw framework"
- **Show, don't tell** — If they allow screen share, demo the website live
- **Texas pride** — Emphasize "local Texas business, built in Hilltop Lakes"
- **Soft CTA** — "Book a free discovery call" not "buy now"
