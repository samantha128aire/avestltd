**What you need:**

#### **First-Time Setup (5 minutes)**

1. **Connect hardware:**
   - Plug in power cable
   - Connect HDMI cable to monitor
   - Connect keyboard & mouse (USB or pair Bluetooth)
   - Connect Ethernet cable (optional, WiFi is fine)

2. **Power on:**
   - Press power button on back of Mac Mini
   - Wait 30-60 seconds for boot
   - You should see macOS login screen on monitor

3. **Log in:**
   - Username: [Provided in your handoff document]
   - Password: [Provided in your handoff document]
   - (You can change this later)

4. **Verify OpenClaw is running:**
   - Open Terminal (Applications > Utilities > Terminal)
   - Type: `openclaw gateway status`
   - Should say "Gateway: running"

5. **Test remote access:**
   - On your laptop/iPad/phone, open Chrome
   - Go to: remotedesktop.google.com/access
   - Sign in with your Google account
   - Click your Mac Mini
   - Enter PIN: [Provided in handoff document]
   - You should see your Mac Mini screen!

**Congratulations!** Your Mac Mini is ready to use.

### Remote Access Instructions

How to access your Mac Mini from anywhere.

#### **Chrome Remote Desktop (Primary Method)**

**From Computer (Windows/Mac/Linux):**
1. Open Chrome browser
2. Go to: remotedesktop.google.com/access
3. Sign in with your Google account
4. Click your Mac Mini name
5. Enter PIN
6. You're in!

**From iPad/iPhone:**
1. Install "Chrome Remote Desktop" app from App Store
2. Open app, sign in with Google account
3. Tap your Mac Mini
4. Enter PIN
5. Use touchscreen to control

**From Android:**
1. Install "Chrome Remote Desktop" app from Play Store
2. Same as iPhone steps

**Tips:**
- Enable "Full screen" for better experience
- Use keyboard shortcuts (Cmd+C, Cmd+V work normally)
- Disconnect when done (click "Stop Sharing")

#### **macOS Screen Sharing (Mac-to-Mac Only)**

**If you have a Mac:**
1. Open Finder
2. Go > Connect to Server (Cmd+K)
3. Enter: `vnc://[Your-Mac-Mini-IP-Address]`
4. Enter username & password
5. Faster/smoother than Chrome Remote Desktop on local network

**Find your Mac Mini's IP:**
- System Settings > Network > Click WiFi/Ethernet > IP address shown

### Troubleshooting

Common issues and how to fix them.

#### **OpenClaw Not Responding**

**Symptoms:** OpenClaw doesn't reply to messages/emails

**Solutions:**
1. Check if OpenClaw is running:
   - Remote into Mac Mini
   - Open Terminal
   - Type: `openclaw gateway status`
   - Should say "running"

2. If not running, restart:
   - Type: `openclaw gateway restart`
   - Wait 30 seconds
   - Test with simple query

3. Check logs for errors:
   - Type: `tail -n 50 /tmp/openclaw/openclaw-*.log`
   - Look for error messages
   - Send to support if unclear

4. Still not working? Contact support.

#### **Remote Access Not Working**

**Symptoms:** Can't connect to Mac Mini remotely

**Solutions:**
1. Verify Mac Mini is powered on:
   - Check if monitor shows anything
   - Try pinging: In Terminal on your computer, type: `ping [Mac-Mini-IP]`

2. Check internet connection:
   - Mac Mini needs internet to be remotely accessible
   - Verify WiFi/Ethernet connected

3. Verify Chrome Remote Desktop running:
   - Remote in via local screen sharing (if possible)
   - Open Chrome > remotedesktop.google.com/access
   - Verify Mac Mini listed as "Online"

4. Try TeamViewer backup:
   - Open TeamViewer on your device
   - Enter TeamViewer ID: [Provided in handoff doc]
   - Enter password: [Provided in handoff doc]

5. Still can't connect? Call/text support: (508) 922-9086

#### **Integration Not Working (Gmail, Calendar, etc.)**

**Symptoms:** OpenClaw can't read email, calendar, etc.

**Solutions:**
1. Re-authenticate the service:
   - Remote into Mac Mini
   - Open Terminal
   - For Gmail: `gog auth add [your-email@gmail.com]`
   - Follow browser authentication flow
   - Test: `gog gmail search 'is:unread' --max 5`

2. Check API limits:
   - Some services have daily limits
   - Wait 24 hours and try again

3. Verify account permissions:
   - Go to Google Account settings
   - Check "Third-party apps with account access"
   - Ensure OpenClaw/gog has access

4. Contact support if persistent.

#### **Performance Issues (Slow, Laggy)**

**Symptoms:** Mac Mini running slowly, OpenClaw responses delayed

**Solutions:**
1. Check resource usage:
   - Open Activity Monitor (Applications > Utilities)
   - Look for high CPU or RAM usage
   - If consistently >80%, contact support

2. Restart Mac Mini:
   - Apple menu > Restart
   - Wait 2-3 minutes
   - Test performance

3. Check for updates:
   - Apple menu > System Settings > General > Software Update
   - Install if available (or contact support first)

4. Free up disk space:
   - Check storage: Apple menu > About This Mac > Storage
   - Should have >20GB free
   - Delete large files if needed
