## Detailed Phase Breakdown

### **PHASE 1: Lead Generation & Initial Contact**

**Touchpoints:**
- Website visitor clicks "Schedule Free Consultation" → Calendly
- Friend referral texts/calls directly
- Social media DM (LinkedIn, Facebook)
- Email inquiry (samantha@avestltd.com)

**Samantha's Role (AI):**
- Monitor email for inquiries (respond within 4 hours)
- Send Calendly booking confirmation
- Send pre-call questionnaire (optional, 5 questions)

**Chance's Role:**
- Review pre-call notes (if provided)
- Prepare for discovery call

**Timeline:** Immediate to 48 hours

---

### **PHASE 2: Discovery Call (30 minutes)**

**Goals:**
1. ✅ Understand their business/workflow
2. ✅ Identify pain points (email, calendar, tasks, etc.)
3. ✅ Determine if AvestAI is a good fit
4. ✅ Set expectations (cost, timeline, process)
5. ✅ Build trust and rapport

**Questions to Ask:**
- "What does a typical morning look like for you?"
- "What tasks eat up most of your time?"
- "If you could automate one thing, what would it be?"
- "Have you tried any AI tools before? What was your experience?"
- "Do you have a Mac Mini, or would you need one included?"
- "What's your comfort level with technology?"

**Decision Points:**

**✅ GOOD FIT if:**
- Affluent professional (values time over money)
- Clear pain points we can solve
- Willing to invest $695-1,295
- Mac user or willing to buy Mac Mini
- Not overly technical (doesn't want to DIY)
- Reasonable expectations

**❌ NOT A FIT if:**
- Price shopper ("Can you do it for $200?")
- Wants to DIY and just needs advice
- Expects AI to be perfect/human-level
- Has unrealistic timelines ("I need it tomorrow")
- Doesn't trust cloud or local AI (fears all AI)
- Too technical (would rather set it up themselves)

**Outcome:**
- **If good fit:** Send intake form + Terms of Service
- **If not a fit:** Be honest, recommend alternatives (DIY guide, ClawFor.biz), stay friendly

**Timeline:** 30 minutes

---

### **PHASE 3: Intake Form & Terms of Service**

**Samantha sends via email:**
1. **Customer Intake Form** (Google Form or fillable PDF)
   - Contact info
   - Business type
   - Desired integrations (email, calendar, messaging, etc.)
   - Use cases (what do you want AI to do?)
   - Existing accounts (Gmail, iCloud, etc.)
   - Mac Mini preference (own vs. purchase)

2. **Terms of Service** (PDF)
   - Review and acknowledge

**Chance reviews:**
- Check for any red flags
- Identify custom integrations needed
- Estimate complexity (standard vs. custom work)

**Timeline:** 24-48 hours for customer to complete

---

### **PHASE 4: Proposal & Quote**

**Samantha generates (from template):**

**Quote includes:**
- Package selected (Basic $695, Standard $1,295, Premium $1,795)
- What's included:
  - Mac Mini M4 (if applicable)
  - OpenClaw installation & configuration
  - [X] integrations (list specific: Gmail, Calendar, iMessage, etc.)
  - Security hardening
  - 30-day hand-holding support
  - Remote access setup (Chrome Remote Desktop + TeamViewer backup)
- Timeline: 5-7 business days from deposit
- Payment terms: 50% deposit to begin, 50% before delivery
- Next steps: Reply to accept, send deposit via [Zelle/check/credit card]

**Timeline:** 24 hours to send proposal

---

### **PHASE 5: Deposit & Queue**

**Customer sends 50% deposit:**
- Zelle (preferred, instant)
- Check (slower, wait for clearance)
- Credit card (3% processing fee)

**Samantha:**
- Send receipt
- Confirm timeline ("Setup begins [date], delivery estimated [date]")
- Add to setup queue

**Chance:**
- Order Mac Mini (if Standard/Premium) - Amazon Prime 2-day
- Plan integration strategy based on intake form

**Timeline:** Immediate upon receipt

---

### **PHASE 6: Setup (5-7 Business Days)**

#### **Day 0: Deposit Received**
- Order Mac Mini (if needed)
- Review intake form thoroughly
- Plan integration strategy
- Document in customer folder

#### **Day 1-2: Hardware Arrival & Initial Config**
- Unbox Mac Mini
- Initial macOS setup
- Create user account (customer's name)
- Enable remote access (Chrome Remote Desktop)
- Install Homebrew
- System updates

#### **Day 3-4: OpenClaw Installation & Security**
- Install OpenClaw (`curl -fsSL https://openclaw.ai/install.sh | sh`)
- Security hardening:
  - Firewall enabled
  - File permissions set
  - No unnecessary services exposed
- Install vetted CLIs (gog, imsg, etc. per intake form)
- Configure AI provider (Anthropic Claude recommended)

#### **Day 4-5: Integrations & Configuration**
- Connect integrations per intake form:
  - Gmail (via gog)
  - Calendar (via gog or Apple Calendar)
  - iMessage (via imsg)
  - Industry-specific (aviation weather, agriculture data, etc.)
- Configure use cases:
  - Morning brief (if requested)
  - Email digest (if requested)
  - Specific workflows from intake form
- Write custom SKILL.md files if needed

#### **Day 5-6: Testing & Documentation**
- Test all integrations (send test email, check calendar, etc.)
- Test remote access from external network
- Run security scan (no exposed ports)
- Document setup in customer folder:
  - What integrations installed
  - API keys used (stored securely)
  - Custom configurations
  - Known issues/limitations

#### **Day 6-7: Pre-Delivery Check**
- Final walkthrough (pretend you're the customer)
- Test morning brief (if configured)
- Verify remote access works from phone/laptop
- Screenshot everything working (for records)

**Communication during setup:**
- Day 3: "Setup is progressing well, Mac Mini configured"
- Day 5: "Integrations connected, testing in progress"
- Day 7: "Setup complete! Ready for final payment and delivery"

---

### **PHASE 7: Final Payment & Delivery**

**Request final payment (50%):**

**Samantha sends:**
- Invoice with setup summary:
  - Package: [Standard $1,295]
  - What was configured (list all integrations)
  - Timeline achieved (5 days)
  - Remote access details (will be provided after payment)
- Payment instructions
- Next steps: "Once payment received, we'll ship Mac Mini / provide remote access"

**After payment received:**

**If shipping Mac Mini (Standard/Premium):**
- Package securely with:
  - Power cable
  - Setup guide (1-page quick start)
  - Remote access instructions
  - Support contact info
- Ship via UPS/FedEx with tracking
- Send tracking number + estimated delivery
- Email: "Your Mac Mini is on the way!"

**If customer's Mac Mini (Basic):**
- Provide remote access credentials via email
- Schedule onboarding call (next 24-48 hours)

**Timeline:** 1-2 days for shipping, immediate for remote-only

### **PHASE 8: Onboarding Call (30-45 minutes)**

**Schedule within 24 hours of delivery/access**

**Goals:**
1. ✅ Ensure customer can access Mac Mini remotely
2. ✅ Walkthrough basic usage
3. ✅ Demo configured use cases
4. ✅ Answer questions
5. ✅ Set support expectations

**Agenda:**

**1. Remote Access Setup (10 min)**
- Walk through Chrome Remote Desktop connection
- Verify they can see Mac Mini desktop
- Show TeamViewer backup (if primary fails)
- Test on both computer and phone

**2. OpenClaw Basics (10 min)**
- How to talk to AI (via iMessage or configured channel)
- Show memory system (it remembers context)
- Demonstrate one simple task ("Check my calendar tomorrow")

**3. Configured Use Cases (15 min)**
- Demo morning brief (if configured)
- Show email digest (if configured)
- Walk through any custom workflows
- Explain how to adjust/customize

**4. Safety & Best Practices (5 min)**
- "Never install skills from ClawHub without asking us first"
- If AI suggests something strange, text us before doing it
- Keep Mac Mini plugged in and connected to internet
- macOS updates will happen automatically

**5. Support Period Expectations (5 min)**
- 30 days of hand-holding support included
- We'll check in proactively (Day 3, 7, 14, 25)
- You can text/call/email anytime during business hours
- Response time: Within 4 hours during business hours (9 AM - 5 PM CST)

**6. Q&A (time permitting)**

**Outcome:** Customer feels confident and supported

### **PHASE 10: Post-Support Relationship**

**Day 30+: Four Customer Paths:**

#### **Path A: Happy & Self-Sufficient** (60% of customers)

**What they say:** "I'm good! Everything works great. I'll reach out if I need help."

**Your response:**
> "That's great to hear! I'm glad it's working well for you.
>
> Remember - you have ONE free 30-min session banked (valid 1 year) since you didn't use much support. Use it anytime.
>
> I'll check in quarterly just to say hi and share any cool new features. And if you ever know anyone who needs this, send them my way!"

**Ongoing touchpoints:**
- Quarterly check-in (every 3 months)
- Share OpenClaw updates/new features
- Ask for referral after 90 days
- Request testimonial after 90 days

---

#### **Path B: Needs Ongoing Support** (25% of customers)

**What they say:** "I still have questions sometimes. What's the support plan?"

**Your response:**
> "Happy to continue supporting you! Monthly support is $100/month and includes:
> - Email/text support (response within 4 hours)
> - Up to 1 hour of remote assistance per month
> - Proactive updates about new features
> - Priority for custom requests
>
> Sound good?"

**If they sign up:**
- Monthly check-in (first week of month)
- Track support hours (bill overage at $150/hour)
- Build relationship (they're recurring revenue)
- Upsell custom features as appropriate

---

#### **Path C: Wants to Add Features** (10% of customers)

**What they say:** "Can you add [X integration]? I'd pay extra for that."

**Your response:**
> "Absolutely! Custom integrations are $150/hour. For [X integration], I estimate [Y hours].
>
> Would you like me to scope it out and send you a quote?"

**Process:**
1. Scope work (1-2 hours to research/plan)
2. Send quote (hours estimate + total cost)
3. Customer approves
4. Build integration (bill actual hours)
5. Test & deliver
6. Add to support plan if they want ongoing help

---

#### **Path D: Unhappy / Has Issues** (5% of customers)

**What they say:** "This isn't working like I expected" or "I'm frustrated with [X]"

**Your response:**
> "I'm really sorry to hear that. Let's get this fixed. Can we schedule a call today or tomorrow?
>
> I want to make sure you're getting value from this. If we can't get it working the way you need, we'll figure out a solution together."

**Troubleshooting steps:**
1. Schedule immediate call (same day if possible)
2. Listen to their frustration (don't get defensive)
3. Identify root cause:
   - Misconfigured? Fix it.
   - Unrealistic expectations? Reset them.
   - Missing feature? Build it or find workaround.
4. Make it right:
   - Extend support period if it's your fault
   - Build missing feature at no charge (if reasonable)
   - If truly not a fit, discuss options (but no refund per terms)

**Goal:** Turn unhappy customer into advocate through exceptional service

## Decision Trees

### **Discovery Call: Is This a Good Fit?**

```
Does customer have clear pain points? (email, calendar, tasks, etc.)
    │
    ├─ NO → Probe deeper: "What frustrates you about your current workflow?"
    │          │
    │          ├─ Still no clear pain → NOT A FIT (they don't need it)
    │          └─ Pain emerges → Continue
    │
    └─ YES → Is customer willing to invest $695-1,295?
               │
               ├─ NO / "That's too expensive" → NOT A FIT (price shopper)
               │
               └─ YES / "That's reasonable" → Does customer want hands-on or hands-off?
                          │
                          ├─ Hands-on (wants to DIY, just needs guidance) → NOT A FIT
                          │
                          └─ Hands-off (wants you to do everything) → GOOD FIT ✅
```
