\section*{\textcolor{darkblue}{EXECUTIVE SUMMARY}}

PropsUAV Drone Launch Platform is a complete, buildable system for operating high-capacity agricultural spray drones from a mobile trailer-based launch facility. All technical specifications, drawings, and documentation are included in this package.

\subsection*{Key Specifications}
\begin{itemize}
  \item \textbf{Platform Elevation:} 30-36'' above existing roof
  \item \textbf{Liquid Storage:} 550 gallons (400-450 gal recommended operating)
  \item \textbf{Trailer Dimensions:} 163'' + 23'' V-nose, 84'' wide, 83'' interior height
  \item \textbf{Payload Capacity:} 4,700 lbs available
  \item \textbf{Build Timeline:} 9-14 weeks
  \item \textbf{Cost Estimate:} \$11,770--\$20,400 (fabrication + installation only)
\end{itemize}

\subsection*{Design Highlights}
\begin{itemize}
  \item [\checkmark] Full 84'' platform width for stable drone launch/landing
  \item [\checkmark] Elevated pilot station with optimal visibility (120-126'' eye level)
  \item [\checkmark] Integrated 550-gallon storage (400-450 gal recommended operating volume)
  \item [\checkmark] Welded steel construction (low maintenance, high durability)
  \item [\checkmark] Protected pilot area (railing + mesh screen)
  \item [\checkmark] Quick-access fill system (center-front nozzle)
  \item [\checkmark] Detailed fabrication drawings ready for fabricator quotes
\end{itemize}

\section{ASSEMBLY SEQUENCE}

\subsection{Phase 1: Trailer Preparation}
\begin{enumerate}
  \item Clean and prepare roof surface
  \item Mark attachment points (four corners)
  \item Install backing plates if needed
\end{enumerate}

\subsection{Phase 2: Platform Frame Fabrication (Off-Site)}
\begin{enumerate}
  \item Cut and fit aluminum extrusion
  \item Assemble main frame (rectangular base)
  \item Weld gusset plates at attachment points
  \item Install decking and safety railing
  \item Powder coat all components
\end{enumerate}

\subsection{Phase 3: Stair Fabrication (Off-Site)}
\begin{enumerate}
  \item Cut and bend steel tube for stringers
  \item Fit and tack all step treads
  \item Full penetration welds on all connections
  \item Verify step rise/run tolerance (\textpm{}1/4'')
  \item Fabricate and weld mounting brackets
\end{enumerate}

\subsection{Phase 4: Platform Installation}
\begin{enumerate}
  \item Lift platform frame onto trailer roof (crane required)
  \item Align four corner attachment points
  \item Tack-weld gusset plates to roof beams
  \item Perform final alignment (level check)
  \item Full penetration welds on all four corners
  \item Mount and weld stair assembly
  \item Install safety railing and mesh screens
\end{enumerate}

\subsection{Phase 5: Utilities Integration}
\begin{enumerate}
  \item Install generator in V-nose compartment
  \item Route and secure exhaust pipe
  \item Install pump and plumbing connections
  \item Pressure test all hose connections (50 PSI minimum)
  \item Install fill nozzle on platform
  \item Route electrical conduit for power
\end{enumerate}

\subsection{Phase 6: Testing \& Safety Verification}
\begin{enumerate}
  \item Static load test on platform (500 lbs minimum)
  \item Full system test with water at operating pressure
  \item Generator operation and exhaust isolation check
  \item Stair integrity and load rating verification
  \item Tie-down anchor strength verification
\end{enumerate}

\section{OPERATIONAL SAFETY NOTES}

\textcolor{red}{\textbf{⚠ Confined Space Risk:}} Interior may accumulate toxic fumes. Generator exhaust MUST exit through right wall. Operator must use in open air only.

\textcolor{red}{\textbf{⚠ Load Capacity:}} Platform is rated for pilot + drone (\textasciitilde{}200 lbs max additional equipment). Do NOT exceed designed weight.

\textcolor{red}{\textbf{⚠ Electrical Hazard:}} All 120V/240V circuits must have proper GFCI protection. Generator bonding required.

\textcolor{red}{\textbf{⚠ Wind Loading:}} Platform acts as a sail. DO NOT operate in sustained winds >20 mph.

\textcolor{red}{\textbf{⚠ Tie-Down Requirements:}} Platform and drone MUST be secured during transport.
