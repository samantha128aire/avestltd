## TIPS FOR USING THIS TEMPLATE

1. **Always start with character consistency** — MagicLight needs to see Sophia, Piper, etc. described the same way every time

2. **Be specific about aircraft** — Don't say "small plane" — say "yellow Piper J-3 Cub, high-wing tail dragger"

3. **Reference the Production Bible** — Keep it open while filling in this template

4. **Environment matters** — Always mention grass runway, metal hangars, Texas setting, big sky

5. **Lighting sets mood** — Golden morning = warm/hopeful, midday = bright/active, evening = reflective/peaceful

6. **Piper's ears are critical** — Mention ear position in emotional scenes (back when worried, up when happy)

7. **Prohibited elements** — Explicitly state "No jets, no helicopters, no military aircraft visible on ground"

8. **Test for consistency** — If you're unsure whether something matches previous episodes, check the Production Bible
