### 2. Obtain DJI Documentation
You need to download and include these DJI documents with your petition:

- **DJI T40 Battery User Guide**
- **DJI T40 Charger User Guide**

Download from: https://www.dji.com/support/product/t40

### 3. Register Aircraft (After Exemption Approved)
- You'll need an N-number for the T40 (over 55 lbs)
- Register at: https://www.faa.gov/licenses_certificates/aircraft_certification/aircraft_registry
- Display N-number on aircraft

### 4. Submit Petition
**Where:** https://www.regulations.gov/commenton/FAA-2007-0001-0001

**When:** At least 120 days before you want to start operations

**How:**
1. Convert documents to PDF
2. Mark confidential documents as "CONFIDENTIAL" when uploading
3. Submit online at regulations.gov
4. Keep confirmation receipt
5. Wait for FAA response (3 weeks to 120 days)

### 5. After Exemption Approval
Once you receive your exemption number:
1. Complete FAA Form 8710-3
2. Submit to: UAS137Certificates@faa.gov
3. Include your exemption number
4. Wait for Agricultural Aircraft Operator Certificate (AAOC)
