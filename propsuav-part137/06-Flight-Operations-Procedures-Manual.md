## 1. INTRODUCTION

### Purpose
This manual provides standardized procedures for conducting agricultural UAS operations for Avest Ltd. Strict adherence to these procedures is mandatory for all operations.

### Applicability
This manual applies to:
- All agricultural UAS operations under Part 137
- All pilots acting as PIC
- All crew members
- All DJI Agras T40 operations

### Authority
Operations are conducted under:
- 14 CFR Part 107
- 14 CFR Part 137
- FAA Exemption (to be issued)
- State of Texas regulations

## 2. REGULATORY COMPLIANCE

### 2.1 Operating Limitations

**Airspace:**
- Class G airspace only
- No operations in controlled airspace without authorization
- Check NOTAMs daily for TFRs

**Altitude:**
- Maximum 200 feet AGL
- Minimum altitude: as required for safe operation

**Visibility:**
- Minimum 3 statute miles
- Pilot must be able to see aircraft with unaided vision

**Time:**
- Daylight operations only
- No operations before civil dawn or after civil dusk

**Weather:**
- No operations in clouds
- No operations in precipitation
- Wind limit: 15 mph

---

### 2.2 Aircraft Requirements

**Registration:**
- Aircraft registered with FAA (N-number)
- Registration certificate on file
- Registration markings visible

**Airworthiness:**
- Aircraft maintained per manufacturer specifications
- Maintenance logged
- Annual inspection current

**Equipment:**
- Aircraft equipped with:
  - GPS
  - Obstacle avoidance
  - Return-to-home capability
  - Real-time telemetry
  - Automatic spray shutoff

---

### 2.3 Personnel Requirements

**Pilot in Command:**
- Valid Part 107 Remote Pilot Certificate
- Completed Avest Ltd training program
- Current endorsement from Chief Supervisor
- Medical fitness (self-certification)

**Visual Observer (if used):**
- Completed crew training
- Effective communication with PIC
- Unaided visual contact with aircraft

**Crew Members:**
- Trained in specific duties
- Familiar with emergency procedures
- Appropriate PPE for chemical operations

## 4. AIRCRAFT OPERATIONS

### 4.1 Pre-Flight Procedures

**Administrative:**
1. Review mission requirements
2. Check weather forecasts
3. Verify NOTAM information
4. Confirm aircraft availability
5. Ensure required documents available
6. Brief crew on mission

**Aircraft Inspection:**
1. Use official checklist (see Manual 02)
2. Inspect airframe for damage
3. Check all motors and propellers
4. Verify battery health and charge
5. Test flight control systems
6. Check GPS signal strength
7. Verify obstacle avoidance systems
8. Test spray/spread system
9. Confirm communications

**Mission Planning:**
1. Map field boundaries
2. Identify obstacles
3. Plan flight path
4. Set application parameters
5. Calculate chemical requirements
6. Identify refill locations
7. Designate emergency landing zones

---

### 4.2 Engine Start and System Check

**Startup Sequence:**
1. Install fully charged battery
2. Power on remote controller
3. Connect mobile device
4. Launch DJI Agras app
5. Power on aircraft
6. Wait for system initialization
7. Check GPS satellite count (≥10)
8. Verify home point set correctly
9. Check compass calibration status
10. Review system status indicators
11. Set return-to-home altitude
12. Verify geofencing active

**System Checks:**
□ All motors responding to throttle  
□ Controls responsive and correct  
□ Telemetry data receiving  
□ Video link stable (if applicable)  
□ Battery voltage normal  
□ GPS accuracy acceptable  
□ Obstacle avoidance active  
□ No error messages  
□ Return-to-home tested

---

### 4.3 Takeoff Procedures

**Pre-Takeoff:**
1. Verify takeoff area clear
2. Brief crew on takeoff direction
3. Confirm wind direction and speed
4. Announce "Clear to start motors"
5. Arm aircraft
6. Allow motors to stabilize

**Takeoff:**
1. Announce "Takeoff"
2. Gradual throttle application
3. Lift to 5-foot hover
4. Check aircraft stability
5. Check control response
6. Check telemetry data
7. Proceed with mission if all normal

**Rejected Takeoff:**
- Land immediately if:
  - Unusual vibration
  - Motor sound abnormal
  - Control inputs not responsive
  - Warning messages appear
  - Any safety concern

---

### 4.4 In-Flight Procedures

**Normal Operations:**
- Maintain VLOS at all times
- Monitor telemetry continuously
- Scan for obstacles
- Check battery status regularly
- Watch for weather changes
- Maintain situational awareness
- Stay within operating area

**Visual Observer Procedures:**
- Maintain visual contact with aircraft
- Scan for obstacles and hazards
- Monitor for other aircraft
- Watch for unauthorized personnel
- Communicate clearly with PIC
- Have no other duties during flight

**Lost Visual Contact:**
- Announce "Lost visual"
- Reduce speed or hover
- Climb altitude if safe
- Visual observer provides guidance
- Regain visual before continuing
- Land if visual cannot be reestablished

---

### 4.5 Landing Procedures

**Approach:**
1. Select suitable landing zone
2. Announce "Preparing to land"
3. Verify area clear
4. Reduce altitude gradually
5. Adjust for wind
6. Monitor descent rate
7. Prepare for touchdown

**Landing:**
1. Announce "Landing"
2. Soft touchdown
3. Reduce throttle to idle
4. Allow aircraft to settle
5. Disarm motors
6. Disconnect battery (if ending operations)

**Go-Around:**
Execute if:
- Landing zone not clear
- Unexpected obstacle
- Aircraft unstable
- Any safety concern

---

### 4.6 Post-Flight Procedures

**Immediate:**
1. Disconnect battery
2. Secure aircraft
3. Clean spray system (if used)
4. Inspect aircraft for damage
5. Note any discrepancies
6. Log flight time

**Documentation:**
1. Complete flight log
2. Record chemical usage
3. Document area treated
4. Note any incidents or issues
5. Update maintenance records

**Aircraft Securing:**
1. Clean exterior
2. Remove residues
3. Inspect for wear/damage
4. Store in secure location
5. Charge batteries appropriately
6. Prepare for next flight

## 5. MISSION PLANNING

### 5.1 Site Survey

**Pre-Operation Survey:**
□ Identify field boundaries  
□ Locate power lines  
□ Map trees and structures  
□ Note water sources  
□ Identify sensitive areas  
□ Check for people/livestock  
□ Assess ground conditions  
□ Identify access points

**Documentation:**
- GPS coordinates of field corners
- Obstacle locations and heights
- Photos of site
- Hazards identified
- Buffer zones established
- Property owner contact info

---

### 5.2 Flight Planning

**Mission Parameters:**
- **Spray Width:** 7 meters (typical)
- **Flight Speed:** 7 m/s (typical, adjust for conditions)
- **Flight Altitude:** 3 meters AGL (typical)
- **Swath Overlap:** 10-20% (ensure coverage)
- **Application Rate:** Per chemical label

**Route Planning:**
1. Map field in DJI Agras app
2. Set field boundaries
3. Mark obstacles/no-fly zones
4. Plan flight direction (consider wind)
5. Set waypoints
6. Review route
7. Calculate:
   - Total area
   - Chemical needed
   - Number of tanks
   - Estimated flight time
   - Battery changes needed

---

### 5.3 Resource Planning

**Chemical Calculation:**
```
Total Chemical = (Area × Application Rate) + 10% overage
Number of Loads = Total Chemical ÷ Tank Capacity
```

**Battery Planning:**
```
Flight Time per Load = 6-7 minutes (full tank)
Batteries Needed = (Number of Loads × 1.2) rounded up
```

**Personnel:**
- PIC
- Visual observer (if needed)
- Ground crew for mixing/loading
- Spotter (if required)

**Equipment:**
- Aircraft
- Batteries (charged)
- Remote controller
- Mixing equipment
- PPE
- Spill kit
- First aid kit

## 6. FLIGHT PROCEDURES

### 6.1 Normal Procedures

**Automated Mission:**
1. Complete pre-flight
2. Start mission in app
3. Takeoff
4. Monitor automated flight
5. Aircraft follows programmed route
6. Land when tank empty or battery low
7. Refill/recharge
8. Resume mission
9. Continue until field complete

**Manual Flight:**
1. Complete pre-flight
2. Takeoff and hover
3. Proceed to start of swath
4. Initiate spray
5. Fly swath at constant altitude/speed
6. End spray at swath end
7. Reposition for next swath
8. Repeat until field complete
9. Return and land

---

### 6.2 Emergency Procedures

**Loss of GPS:**
1. Switch to manual control (if not already)
2. Maintain stable hover
3. Cease spray operations
4. Assess situation
5. If GPS not recovered in 30 seconds:
   - Manually fly to safe landing zone
   - Land immediately

**Low Battery Warning:**
1. Note current position
2. Complete current swath if battery adequate
3. If not, cease spray immediately
4. Return to takeoff point
5. Land
6. Replace battery before continuing

**Motor Failure:**
1. Announce "Motor failure"
2. Attempt to maintain control
3. Select nearest safe landing area
4. Cease spray immediately
5. Prepare for hard landing
6. Land with minimum speed
7. Secure area after landing

**Lost Link:**
1. Aircraft will automatically:
   - Hover for 3 seconds
   - Initiate return to home
   - Land at home point
2. PIC actions:
   - Attempt to regain link
   - Monitor aircraft
   - Clear landing area
   - Be ready to regain control

**Fly-Away:**
1. Attempt to regain control
2. Activate return-to-home
3. Note last known position
4. Call out aircraft heading/altitude
5. Warn people in flight path
6. Call 911 if aircraft threatens people/property
7. Track aircraft position if possible
8. Document event

---

### 6.3 Abnormal Procedures

**High Winds Encountered:**
1. Assess wind speed
2. If ≥15 mph:
   - Cease spray operations
   - Return and land
   - Suspend operations
3. If <15 mph but gusty:
   - Reduce speed
   - Increase altitude slightly
   - Widen buffer zones
   - Continue with caution

**Unexpected Obstacle:**
1. Stop forward progress
2. Hover or climb
3. Assess situation
4. Reroute around obstacle
5. Update field map
6. Continue mission

**Unauthorized Person in Area:**
1. Cease spray immediately
2. Hover clear of person
3. Announce to person to clear area
4. Visual observer directs person out
5. If person does not leave:
   - Land aircraft
   - Secure area
   - Request person leave
   - Do not continue until clear

## 7. AGRICULTURAL OPERATIONS

### 7.1 Chemical Handling

**Mixing:**
1. Review product label
2. Calculate required amount
3. Don appropriate PPE
4. Mix in well-ventilated area
5. Follow label mixing instructions
6. Filter mixture into tank
7. Record mixture details
8. Clean mixing equipment
9. Dispose of waste properly

**Loading:**
1. Verify aircraft ready
2. Fill tank through filter
3. Secure tank cap
4. Check for leaks
5. Prime spray system
6. Test spray pattern on ground
7. Document load (chemical, amount, time)

**Application:**
1. Follow mission plan
2. Maintain proper altitude
3. Maintain proper speed
4. Ensure uniform coverage
5. Monitor spray operation
6. Adjust for conditions
7. Document application (area, time, conditions)

---

### 7.2 Application Techniques

**Spray Operations:**
- **Swath Width:** 7 meters
- **Overlap:** 10-20%
- **Altitude:** 3 meters AGL (typical)
- **Speed:** 7 m/s (adjust for conditions)
- **Droplet Size:** Per label requirements

**Factors Affecting Application:**
- Wind speed/direction
- Temperature
- Humidity
- Crop canopy
- Chemical properties

**Quality Indicators:**
- Uniform coverage
- Minimal drift
- Proper droplet size
- Accurate application rate
- Complete field coverage

---

### 7.3 Calibration

**Flow Rate Calibration:**
1. Fill tank with water
2. Operate spray system
3. Collect output for 1 minute
4. Measure volume
5. Compare to expected rate
6. Adjust if needed
7. Repeat until accurate
8. Document calibration

**Spray Width Verification:**
1. Set up collection cups across swath
2. Single pass over cups
3. Measure collected volume
4. Plot distribution pattern
5. Verify 7-meter effective width
6. Adjust as needed

**Weighing Sensor Check:**
1. Fill tank with known amount
2. Check sensor reading
3. Should match within 2%
4. Recalibrate if needed

## 9. WEATHER OPERATIONS

### 9.1 Weather Evaluation

**Pre-Flight:**
- Check aviation weather
- Review local forecasts
- Assess current conditions
- Make go/no-go decision

**Weather Minimums:**
- Visibility: ≥ 3 statute miles
- Wind: ≤ 15 mph
- Temperature: -4°F to 113°F
- No precipitation
- Daylight only

**Deteriorating Weather:**
- Monitor conditions continuously
- Plan for early termination
- Know safe landing zones
- When in doubt, land

---

### 9.2 Seasonal Considerations

**Summer:**
- High temperatures affect battery performance
- Afternoon thunderstorms common
- Plan morning operations
- Allow batteries to cool

**Winter:**
- Short daylight hours
- Cold affects battery performance
- Warm batteries before flight
- Watch for ice formation

**Spring/Fall:**
- Rapidly changing weather
- Wind gusts common
- Temperature fluctuations
- Monitor forecasts closely
