## Technical Specifications

### For 2D Illustrations:
- **Resolution:** 4000x4000px minimum
- **Format:** PNG with transparent background
- **Style:** Flat, clean, suitable for print
- **Color space:** RGB, vibrant but not oversaturated
- **Use cases:** Activity books, worksheets, website graphics

### For 3D Renders:
- **Resolution:** 4K (3840x2160) or higher
- **Format:** PNG with alpha channel
- **Lighting:** Soft, even, studio-quality
- **Expressions:** Multiple facial expressions needed
- **Poses:** Standing, pointing, excited, thinking, exploring
- **Use cases:** Video content, promotional materials, 3D assets

## Free AI Image Generation Options to Try

### Option 1: Leonardo.ai (FREE TIER)
- **URL:** https://leonardo.ai
- **Free credits:** 150 per day
- **Best for:** Both 2D and 3D styles
- **Process:** Sign up → Select "Creative" or "3D Animation" model → Paste prompts

### Option 2: Ideogram.ai (FREE)
- **URL:** https://ideogram.ai
- **Free:** Limited daily generations
- **Best for:** 2D illustrations, flat design
- **Process:** Sign up → Paste prompt → Generate

### Option 3: Craiyon (Formerly DALL-E Mini) (COMPLETELY FREE)
- **URL:** https://craiyon.com
- **Free:** Unlimited (with ads)
- **Best for:** Quick concepts (lower quality)
- **Process:** Paste prompt → Generate (no signup needed)

### Option 4: Hugging Face Stable Diffusion (FREE)
- **URL:** https://huggingface.co/spaces/stabilityai/stable-diffusion
- **Free:** Unlimited
- **Best for:** Both styles, but requires technical tweaking
- **Process:** Access Space → Enter prompt → Generate
