# AvestAI — Goals

## Primary Goal
Build a profitable AI setup service targeting affluent non-technical professionals (50-70 y/o doctors, executives, business owners) using OpenClaw on Apple Mac Mini hardware.

## Revenue Goals
- Year 1: $40k+ (target 2 customers/month)
- Long-term: $100k+ annual income

## Pricing
- Basic: $695 (setup only)
- Standard: $1,295 (setup + 30-day support)
- Premium: $1,795 (full setup + redundancy + memory architecture + Smart Router + PWA)
- Support: $100/month or $150/hour after 30-day period
- Annual upgrade plan: $200-300/year
- Friend discount: $100 off (applies to Chance's personal network)

## Key Differentiators
1. Data stays home (Mac Mini, not cloud)
2. Smart Model Router (Fast/Smart/Smartest — 40-70% cost savings)
3. Custom PWA interface (no competitor has this)
4. Proper memory architecture (folders + SQLite)
5. Redundancy system (Premium only — backup instance)
6. White-glove setup and support

## Target Market
- Primary: Affluent professionals 50-70 y/o
- Doctors, executives, business owners
- NOT technical DIY users
- NOT competing with ClawFor.biz ($20 European cloud service)
