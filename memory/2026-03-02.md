## Subagent Architecture Discussion — 4:48–4:55 PM CST

- Subagents run independently in background; main session stays free to chat
- Subagents DO cost more (5,000-15,000 extra bootstrap tokens)
- Worth it for: long tasks (10+ min), parallel work
- NOT worth it for: quick tasks under 2-3 min
- Subagents CAN use different models than main session
- Subagents CANNOT access iMessage channel — main session is the only channel-connected one
- Smart router already handles responsiveness (Haiku for quick replies while subagent works)
