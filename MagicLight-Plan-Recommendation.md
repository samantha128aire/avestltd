## WHAT ABOUT CONSISTENCY?

**Your Concern:** "Can MagicLight create consistently across a series with the same name?"

**Answer:** Not automatically, but yes with proper prompting.

**How to Ensure Consistency:**

1. **Use the Master Production Bible** I created
   - Reference same character descriptions every episode
   - Reference same environment descriptions every episode
   - Be specific about aircraft types

2. **Use the Episode Prompt Template** I created
   - Copy/paste character sections exactly
   - Don't vary descriptions between episodes

3. **Save successful renders as references**
   - Once you get a good Sophia render, save the image
   - Upload it as reference for future episodes
   - MagicLight has "character clone" feature

4. **Use same visual style across all episodes**
   - Pick ONE style in Episode 1
   - Don't switch between cartoon/realistic/anime mid-season

5. **Be explicit in every prompt:**
   - "Like in previous episodes, Sophia has long dark hair..."
   - "Consistent with Episode 1, the grass runway..."
   - "Same yellow Piper Cub from Episode 1..."

**User reports suggest:** MagicLight CAN maintain consistency IF you're detailed and explicit in every prompt. The Production Bible + Template I created should handle this.

## ISSUES YOU SPOTTED IN FREE TRIAL

### Problem 1: Jets visible on ground
**Solution:** In every prompt, include:
- "PROHIBITED: No jets visible on ground or in scenes"
- "Only single and twin-engine propeller aircraft"
- List specific allowed aircraft: "Cessna 172, Piper Cub, Kitfox, etc."

### Problem 2: Inconsistent airpark elements
**Solution:** Copy/paste environment description from Production Bible into EVERY episode prompt:
- "Grass runway (NOT paved)"
- "Metal hangars (tan, white, green)"
- "Texas landscape, live oak trees"

### Problem 3: Character appearance varies
**Solution:** 
- Use Character Clone feature (create Sophia once, reuse)
- Include detailed character description in every prompt
- Save first successful character render as reference image
